AI Nerf Gun Aimbot

This project aims to create an autonomous Nerf blaster that detects targets, predicts motion, and actuates aim at targets. (Scrapyard 2025 submission)

Project Overview

AI Nerf Aimbot system: camera feed → detection → tracking → motion prediction → control loop driving pan/tilt servos.

Hardware Platform

Arduino UNO as microcontroller, Tower Pro servos for panning up/down and rotating left/right, and 12V power supply.

  • Compute: Arduino UNO microcontroller
  • Actuation: 2 Tower Pro servo motors driven
  • Power: 12V battery source

Coding

We used Python and Arduino C++ for the AI Nerf Aimbot system. The face tracking API is provided in GitHub.

(Pseudocode Arduino) serial.begin(9600) servoX.attach(pin=3) servoY.attach(pin=5) xAngle = 90 yAngle = 90 servoX.write(xAngle) servoY.write(yAngle) delay(1000) serial.send("DEFAULT," + xAngle + "," + yAngle) while true: if serial.available(): input = serial.read_line().trim() if input == "INIT": serial.send("DEFAULT," + xAngle + "," + yAngle) else: newX, newY = parse_two_ints(input) newX = constrain(newX, SERVO_X_MIN, SERVO_X_MAX) newY = constrain(newY, SERVO_Y_MIN, SERVO_Y_MAX) while xAngle != newX or yAngle != newY: if xAngle < newX: step = 2 if (newX - xAngle) > 5 else 1 xAngle = min(xAngle + step, newX) elif xAngle > newX: step = 2 if (xAngle - newX) > 5 else 1 xAngle = max(xAngle - step, newX) if yAngle < newY: step = 2 if (newY - yAngle) > 5 else 1 yAngle = min(yAngle + step, newY) elif yAngle > newY: step = 2 if (yAngle - newY) > 5 else 1 yAngle = max(yAngle - step, newY) servoX.write(xAngle) servoY.write(yAngle) delay(10) fire()

Full code is on GitHub.
