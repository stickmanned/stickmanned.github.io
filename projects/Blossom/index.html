Blossom - Productivity Chrome Extension

Focus Timer + Garden Simulation • Last updated 2026-04-02

Project Overview

Blossom is a premium Chrome extension that transforms productivity into a rewarding, gamified experience. By combining a high-performance focus timer with an interactive Garden Simulation, Blossom helps users combat digital distractions. Users earn currency through productive work, which is then used to buy, plant, and grow unique tree species in a customizable virtual sanctuary.

Core Features

Focus Timer Engine

Run structured work sessions with reliable timing behavior designed for deep focus.

Garden Simulation

Translate completed focus sessions into visual garden growth for long-term motivation.

Local Progress Economy

Earn in-app currency from productive work and spend it on unique trees and layouts.

Distraction Blocking

Block selected domains during active focus sessions to reduce context-switching.

How It Works

  1. Start a focus timer session and choose your work duration.
  2. Complete sessions to earn coins based on time spent concentrating.
  3. Use coins to purchase and plant new tree species in your virtual sanctuary.
  4. Continue sessions to grow your garden and strengthen productivity habits.

Implementation Details

  • Session data and game state are stored locally for fast performance and privacy-conscious behavior.
  • Distraction blocking evaluates active URLs against a local blocklist only during focus sessions.
  • The reward loop ties each productive session to visible sanctuary growth to reinforce consistency.
